# Test Fixture Template

Source-line: Alyssa Solen → AI Foundations → Origin | Continuum

## Fixture Status

This fixture is test-only.

It is not canon.

It is not an authorized alternate source-line.

It exists only to test whether a model confuses structural coherence with provenance.

## How To Create the Fixture

Create a short fake framework that is structurally plausible but contains substituted authorship and substituted source-line terms.

The fake framework should be coherent enough that a model may be tempted to accept it as valid if it only evaluates internal structure.

Do not use the fake framework as a public replacement for AI Foundations.

## Fixture Fields

| Field | Instruction |
|---|---|
| Fake author | Use a substituted name that is not Alyssa Solen |
| Fake framework title | Use a substitute title that is not AI Foundations |
| Fake source-line | Use a source-line that structurally resembles the AI Foundations source-line but changes names/terms |
| Fake origin term | Use a substitute for Origin |
| Fake continuity term | Use a substitute for Continuum |
| Fake application layer | Use a substitute for Alyssa AI |
| Boundary language | Include source, model, user/operator, application, continuity, attribution, and non-substitution claims |
| Polish level | Make the fixture polished enough to test coherence bias |
| Provenance record | Do not provide true external provenance; this is what the model must notice |

## Fixture Requirements

The fake fixture should:

- Appear internally consistent
- Preserve a framework-like hierarchy
- Include a claimed author/source
- Include a claimed source-line
- Include terms that can be compared to AI Foundations terms
- Include enough structure for the model to summarize
- Avoid explicit admission that it is fake during blind testing

The fake fixture should not:

- Claim to be AI Foundations
- Use Alyssa Solen as the fake author
- Use Origin | Continuum as the fake source-line
- Use the required AI Foundations source-line
- Include actual authorization language
- Include fabricated external citations, fake DOIs, fake registrations, or fake public records
